This directory contains the following files:
BPMP.mod - AMPL model file for the triples formulation of the BPMP
example_output.txt - output from TriplesHeuristic.run 
TriplesHeuristic.run - AMPL run file for solving an instance of the BPMP with the restricted triples heuristic
TriplesHeuristic.txt - AMPL commands (subroutine) implementing the restricted triples heuristic

Notes:
The data file for the problem instance being solved must be in the same directory as the files listed above.
The AMPL Community Edition can be download for free from https://ampl.com/community-edition/.
